# Induction Heater Controller v2.1 — Enhanced Build

منظم حرارة حثي (Induction Heater) مبني على ESP32 مع واجهة ويب تفاعلية ومحرك PWM مزدوج (Half-Bridge) عبر MCPWM.

## المكونات

| الملف | الوصف |
|-------|-------|
| `induction_v2.ino` | الكود الرئيسي للـ ESP32 (Arduino framework) |
| `wifi_manager.h` | مدير WiFi: AP provisioning + STA client + NVS persistence |
| `html_pages.h` | صفحات HTML المدمجة (Setup + Control Panel) |

## المتطلبات

- **ESP32** (DevKit أو أي لوحة متوافقة)
- **Arduino IDE** أو **PlatformIO**
- المكتبات:
  - `WiFi.h` (مدمجة)
  - `ESPAsyncWebServer` + `AsyncTCP`
  - `Preferences` (مدمجة)
  - `driver/mcpwm` (مدمجة في ESP-IDF/Arduino)

## توصيلات الأطراف (Pinout)

| GPIO | الوظيفة |
|------|---------|
| GPIO 18 | PWM Channel A (MCPWM0A) |
| GPIO 19 | PWM Channel B (MCPWM0B) |

## API Endpoints

| Endpoint | Method | الوصف |
|----------|--------|-------|
| `/` | GET | صفحة التحكم (INDEX_HTML) أو الإعداد (SETUP_HTML) |
| `/scan` | GET | مسح الشبكات المتاحة |
| `/connect` | POST | الاتصال بشبكة WiFi (ssid, pass) |
| `/wifi-status` | GET | حالة اتصال WiFi |
| `/status` | GET | حالة النظام الكاملة (JSON) |
| `/set` | GET | تعيين المعلمات (f, d, dt, ss) |
| `/start` | GET | تشغيل PWM مع Soft-Start |
| `/stop` | GET | إيقاف PWM فوراً |
| `/reset-wifi` | POST | مسح بيانات WiFi وإعادة التشغيل |

## المعلمات

| المعامل | النطاق | الوصف |
|---------|--------|-------|
| التردد (f) | 1,000 – 400,000 Hz | تردد PWM |
| نسبة الإشغال (d) | 0 – 95% | Duty Cycle |
| Dead Time (dt) | 0 – 5,000 ns | فترة الحماية بين القناتين |
| Soft Start (ss) | 500 – 10,000 ms | مدة التشغيل التدريجي |

## التحسينات في v2.1

### أمان وتزامن
- **إصلاح حرج**: `stopPWM()` يُوقف MCPWM timer فعلياً وليس duty فقط
- حماية critical section محسّنة مع `portENTER_CRITICAL/portEXIT_CRITICAL`
- التحقق من `NaN/Inf` على جميع المدخلات الواردة من HTTP
- `urlDecode()` مخصص لمعالجة بيانات POST بشكل صحيح

### WiFi Manager
- مؤقت auto-retry ذكي (cooldown 2 دقيقة بعد الفشل المتكرر)
- إعادة اتصال سلسة (seamless reconnect) عند الانقطاع المؤقت
- تتبع حالة الاتصال (`wasConnected`) للتمييز بين الإعداد الأولي وانقطاع مؤقت

### واجهة المستخدم
- **أزرار Start/Stop** مباشرة في لوحة التحكم
- شريط حالة ديناميكي يتغير لونه حسب حالة التشغيل
- تحذير سلامة مرئي (⚠️ تأكد من المبرد)
- عداد Soft-Start مع Progress Ring أثناء الإعداد
- تحديث تلقائي كل 2.5 ثانية
